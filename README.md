# Проект "Путешествие по России"

Путешествие по России - адаптивная вёрстка лендинга. 
Цель проекта - научиться адаптивной вёрстке, продвинутой работе с гридами и флексами.

Стек:

- HTML,
- CSS,
- БЭМ,
- grid
- flexbox
- адаптивная вёрстка
